# Wellness Landing Template - Jarviss AI Component

## Panoramica
Template modulare per landing page di centri benessere con componenti React/Vue ottimizzati.

## Struttura
- `wellness-landing.html`: Template base
- `styles.css`: Fogli di stile modulari
- `wellness.js`: Logica interattiva
- `components/`: Cartella componenti riutilizzabili

## Requisiti
- Framework JS consigliato: React/Vue
- Librerie: TailwindCSS, Alpine.js

## Installazione
1. Clonare repository
2. Installare dipendenze
3. Configurare variabili nel file `.env`

## Personalizzazione
Modificare `config.json` per adattare il template
